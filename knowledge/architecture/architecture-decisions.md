# Architecture Decisions

## Sprint 7: ADRs + Cross-Cutting Concerns

- Orchestrator: Nomad 1.9.x — workload orchestration and scheduling in cluster environment
- Service Discovery: Consul 1.15.x — dynamic service discovery and health checking
- Secret Management: Vault 1.15.x — secure storage and distribution of API keys and credentials
- Model Serving: vLLM 0.5.x — high-throughput LLM serving for local models (Qwen, Nemotron)
- Observability: Langfuse — asynchronous trace accumulation for LLM interactions and routing decisions
- Metrics: Prometheus — collection and storage of system performance metrics
